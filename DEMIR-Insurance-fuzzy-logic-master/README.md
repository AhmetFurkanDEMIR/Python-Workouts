# DEMIR sigorta

Sigorta şirketleri için, bulanık mantık kullanarak yazılmıştır.
Aracın kaza durumu, sürücü sicili ve aracın moturuna göre müşteriye fiyat teklifi yapar.

-------------------------------------

Üçgen (Triangular) Üyelik Fonksiyonu kullanıldı.

0,                     x <= a

(x - a) / (m - a),     a < x <= m

(b - x) / (b - m),     m < x < b

0,                     x >= b
